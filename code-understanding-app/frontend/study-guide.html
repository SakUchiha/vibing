HTML Getting Started

What is HTML?

HTML structures the content of web pages using elements like headings, paragraphs, lists, images, and links.

  • Elements are defined with tags like <h1> and <p>
  • Attributes add extra info like href on links and src on images
<!DOCTYPE html> <html> <head> <title>My Page</title> </head> <body> <h1>Hello World</h1> <p>This is my first web page.</p> </body> </html>
HTML Links and Images

Make Pages Useful

Use anchors for links and the <img> tag for images. Always include alt text for accessibility.

<a href="https://example.com">Visit Example</a> <img src="https://via.placeholder.com/300x200" alt="Sample image">
CSS Styling Basics

What is CSS?

CSS controls the look of your page. You write rules with selectors, properties, and values.

<style> body { font-family: Arial; background: #f5f7fb; } h1 { color: #2c3e50; text-align: center; } .highlight { background: #f39c12; color: white; padding: 4px 8px; } </style>

Attach styles inline, in the <style> tag, or via external CSS files.

CSS Colors and Fonts

Typography and Color

  • Colors: names, hex (#ff6b6b), rgb(rgb(255,107,107))
  • Font properties: font-family, font-size, font-weight
.title { color: #ff6b6b; font-size: 2rem; font-weight: 700; }
JS Interactivity

What is JavaScript?

JavaScript adds behavior. You can handle events, change the DOM, and react to user input.

<button onclick="greet()">Say Hello</button> <script> function greet(){ alert('Hello, JavaScript World!'); } </script>
Practice Mini Challenge

Build a Mini Card

Create a simple profile card with HTML, style it with CSS, and add a JS button that toggles a highlight.

<!DOCTYPE html> <html> <head> <style> .card { width: 280px; margin: 20px auto; padding: 16px; border-radius: 10px; background: white; box-shadow: 0 6px 16px rgba(0,0,0,0.08); } .name { font-weight: 700; font-size: 1.2rem; color: #2c3e50; } .tag { display:inline-block; margin-top:8px; padding: 4px 8px; border-radius:16px; background:#eef2ff; color:#667eea; font-weight:600; } .highlight { background:#667eea; color:white; } button { margin-top: 12px; background:#ff6b35; color:white; border:none; padding:8px 12px; border-radius:20px; cursor:pointer; } </style> </head> <body> <div class="card"> <div class="name">Ada Lovelace</div> <div id="tag" class="tag">Beginner</div> <button onclick="toggleTag()">Toggle Highlight</button> </div> <script> function toggleTag(){ var el = document.getElementById('tag'); el.classList.toggle('highlight'); } </script> </body> </html>
