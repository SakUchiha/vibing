Web Development Study Guide

This comprehensive guide covers the fundamentals of HTML, CSS, and JavaScript. Use it as a reference while learning or as a quick lookup during coding.

HTML Fundamentals

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using markup.

Basic HTML Structure

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My Web Page</title> </head> <body> <h1>Hello World!</h1> <p>This is a paragraph.</p> </body> </html>

Common HTML Tags

Text Elements

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <strong> - Bold text
  • <em> - Italic text

Structure Elements

  • <div> - Container
  • <span> - Inline container
  • <header> - Page header
  • <main> - Main content
  • <footer> - Page footer

Interactive Elements

  • <a href="url"> - Links
  • <img src="image.jpg" alt="description"> - Images
  • <button> - Buttons
  • <input> - Form inputs

Pro Tip

Always include the alt attribute for images to make your website accessible to screen readers.

CSS Styling

What is CSS?

CSS (Cascading Style Sheets) controls the visual appearance of HTML elements. It allows you to style colors, fonts, layouts, and more.

CSS Syntax

selector { property: value; property: value; } /* Example */ h1 { color: blue; font-size: 24px; text-align: center; }

CSS Selectors

Element Selectors

  • h1 - All h1 elements
  • p - All paragraph elements
  • div - All div elements

Class & ID Selectors

  • .classname - Elements with class
  • #idname - Element with ID
  • div.classname - Div with class

Common Properties

  • color - Text color
  • background-color - Background
  • font-size - Text size
  • margin - Outer spacing
  • padding - Inner spacing

Pro Tip

Use the CSS box model to understand how margin, border, padding, and content work together to create spacing.

JavaScript Programming

What is JavaScript?

JavaScript is a programming language that adds interactivity to web pages. It can respond to user actions and manipulate page content.

Basic JavaScript Syntax

// Variables let name = "John"; const age = 25; var city = "New York"; // Functions function greet(name) { return "Hello, " + name + "!"; } // Event handling document.getElementById("myButton").onclick = function() { alert("Button clicked!"); };

JavaScript Concepts

Data Types

  • let num = 42; - Numbers
  • let text = "Hello"; - Strings
  • let flag = true; - Booleans
  • let arr = [1, 2, 3]; - Arrays

DOM Manipulation

  • getElementById() - Get element by ID
  • innerHTML - Change content
  • style.property - Change styles
  • addEventListener() - Add events

Common Events

  • onclick - Mouse click
  • onload - Page loads
  • onchange - Input changes
  • onkeydown - Key pressed

Pro Tip

Always use let or const instead of var for better variable scoping and to avoid common JavaScript pitfalls.

Best Practices

HTML Best Practices

  • Use semantic HTML elements (header, main, article, footer)
  • Always include alt attributes for images
  • Use proper heading hierarchy (h1, h2, h3, etc.)
  • Validate your HTML code

CSS Best Practices

  • Use external stylesheets for better organization
  • Follow a consistent naming convention (like BEM)
  • Use CSS Grid and Flexbox for layouts
  • Write mobile-first responsive designs

JavaScript Best Practices

  • Use meaningful variable and function names
  • Comment your code for complex logic
  • Handle errors gracefully with try-catch
  • Test your code in different browsers
